方法：采用plink的IBS/IBD estimation模块计算样本间的IBS/IBD
备注：
传递一致性（identical-by-descent, IBD）分析:子代中共有的一段DNA片段或共有的等位基因来源于一个共同的祖先（相同等位基因片段是否同源）
状态一致性（identical-by-state, IBS）通过同胞间等位基因比较来推测一段DNA片段或等位基因是否同源


======================================================================
plink.IBD.xls 为分析的结果文件，文件每列介绍如下：

FID1：样本1的家系编号
IID1：样本1的编号
FID2：样本2的家系编号
IID2：样本2的编号
Z0：基因座上，两个样本间共有0个等位基因的可能性
Z1：基因座上，两个样本间共有1个等位基因的可能性
Z2：基因座上，个样本间共有2个等位基因的可能性
PI_HAT：Z0、Z1、Z2的综合打分值。分值为0-1，越接近1，样本间亲缘关系越近
